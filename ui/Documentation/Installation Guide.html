Silvia Coffee Machine - Installation Guide

📋 Prerequisites

Hardware Requirements

  • Microcontroller: Teensy 4.0 or Arduino-compatible board

  • Display: 1920x1080 touchscreen (recommended)

  • Computer: Raspberry Pi 4 or PC with USB port

  • Sensors: PT100, pressure sensor, dual load cells

  • Actuators: Water pump, heater element, solenoid valve

  • Power Supply: 12V/24V for actuators, 5V for electronics

Software Requirements

  • Python: 3.8 or higher

  • Operating System: Windows 10+, Linux, or macOS

  • Arduino IDE: Latest version with library support

  • USB Drivers: For Teensy/Arduino communication

🔧 Hardware Installation

Step 1: Microcontroller Setup

Teensy 4.0 Pin Connections


Temperature Sensor (PT100 via MAX31865):

├── CS → Pin 10

├── MOSI → Pin 11

├── MISO → Pin 12

└── CLK → Pin 13

  

Pressure Sensor (via ADS1115):

├── SDA → Pin 18

└── SCL → Pin 19

  

Load Cells (via HX711):

├── Scale 0 Data → Pin 20

├── Scale 1 Data → Pin 22

└── Clock (shared) → Pin 21

  

Control Outputs:

├── Pump PWM → Pin 9

├── Heater PWM → Pin 15

├── Valve Relay → Pin 8

└── Potentiometer → Pin A0

Wiring Diagram


[Teensy 4.0]

│

├─[MAX31865]─[PT100 Sensor]

├─[ADS1115]─[Pressure Sensor]

├─[HX711 #1]─[Load Cell #1]

├─[HX711 #2]─[Load Cell #2]

├─[PWM]─[Pump Motor Driver]

├─[PWM]─[SSR]─[Heater Element]

└─[Digital]─[Relay]─[Solenoid Valve]

Step 2: Sensor Installation

PT100 Temperature Sensor

  1. Mounting: Install in boiler with thermal compound

  2. Wiring: Use 3-wire configuration

  • Red: RTD+

  • White: RTD-

  • Black: RTD- (jumpered to white)

  1. Shielding: Use shielded cable to prevent interference

Pressure Sensor (Honeywell MIP)

  1. Mounting: Install in water line before group head

  2. Wiring:

  • VCC: 5V

  • GND: Ground

  • OUT: Analog output to ADS1115

  1. Calibration: Ensure zero pressure at startup

Load Cells

  1. Mounting: Install under drip tray or cup platform

  2. Mechanical: Ensure rigid mounting, prevent side loads

  3. Wiring: Follow HX711 color code

  • Red: E+

  • Black: E-

  • White: A-

  • Green: A+

Step 3: Actuator Installation

Water Pump

  1. Power: Connect to 12V/24V supply via PWM driver

  2. Control: PWM signal from Teensy Pin 9

  3. Safety: Install pressure relief valve

  4. Plumbing: Use food-grade tubing and fittings

Heater Element

  1. Power: Connect via Solid State Relay (SSR)

  2. Control: PWM signal from Teensy Pin 15

  3. Safety: Install thermal fuse and proper grounding

  4. Insulation: Ensure proper thermal insulation

Solenoid Valve

  1. Power: 12V/24V via relay module

  2. Control: Digital signal from Teensy Pin 8

  3. Position: Install in water line to group head

  4. Flow Direction: Ensure correct flow direction

Step 4: Power Supply Setup

Power Distribution


Main Power (120V/240V AC)

├── Heater Element (direct, via SSR)

├── 24V Supply

│ ├── Water Pump

│ └── Solenoid Valve

└── 5V Supply

├── Teensy 4.0

├── Sensors

└── Display (if USB powered)

Safety Considerations

  • GFCI Protection: Install GFCI breaker for all AC circuits

  • Grounding: Proper grounding for all metal components

  • Fusing: Individual fuses for each circuit

  • Emergency Stop: Hardware emergency stop switch

💻 Software Installation

Step 1: Arduino Firmware

Install Arduino IDE

  1. Download from arduino.cc

  2. Install Teensyduino add-on for Teensy support

  3. Configure board settings:

  • Board: Teensy 4.0

  • USB Type: Serial

  • CPU Speed: 600 MHz

Install Required Libraries


# Via Arduino Library Manager:

-  Adafruit  MAX31865  library

-  ADS1115_WE  by  Wolfgang  Ewald

-  HX711  Arduino  Library

Upload Firmware

  1. Open silvia_lever_main/silvia_lever_main.ino

  2. Verify pin assignments in config.h

  3. Compile and upload to Teensy

  4. Monitor serial output for “READY” message

Verify Hardware Communication


Expected Serial Output:

Adafruit MAX31865 PT100 Sensor Started!

Calibrating... Keep sensor at zero pressure!

Calibrated Zero Voltage: 0.512

READY

Step 3: System Integration

Auto-Start Configuration (Linux)

Create systemd service:


sudo  nano  /etc/systemd/system/silvia-coffee.service



[Unit]

Description=Silvia Coffee Machine

After=graphical-session.target

  

[Service]

Type=simple

User=pi

WorkingDirectory=/home/pi/silvia-coffee-machine/source/pyqt6

ExecStart=/home/pi/silvia-coffee-machine/source/pyqt6/venv/bin/python run_silvia.py --fullscreen

Restart=always

Environment=DISPLAY=:0

  

[Install]

WantedBy=graphical-session.target

Enable service:


sudo  systemctl  enable  silvia-coffee.service

sudo  systemctl  start  silvia-coffee.service

🆘 Troubleshooting

Common Installation Issues

“DISCONNECTED” Status

Cause: Serial communication failure

Solutions:

  • Check USB cable connection

  • Verify correct COM port in config

  • Ensure Arduino firmware uploaded

  • Try different USB port

Temperature Reading Errors

Cause: PT100 sensor issues

Solutions:

  • Check 3-wire PT100 connections

  • Verify MAX31865 wiring

  • Clear sensor faults

  • Check for electrical interference

Pressure Sensor Not Working

Cause: I2C communication failure

Solutions:

  • Check SDA/SCL connections

  • Verify I2C address (0x48)

  • Check power supply to ADS1115

  • Test with I2C scanner

Scales Not Responding

Cause: HX711 communication issues

Solutions:

  • Check HX711 wiring

  • Verify power supply

  • Check load cell connections

  • Recalibrate scales

⚠️ IMPORTANT SAFETY NOTICE

This system controls high-temperature heating elements and high-pressure water systems. Installation should only be performed by qualified personnel familiar with electrical and plumbing systems. Always follow local electrical and plumbing codes. Test all safety systems before operation.

🔌 ELECTRICAL WARNING

This system involves potentially lethal voltages. Ensure all power is disconnected before working on electrical connections. Use proper lockout/tagout procedures. Install appropriate circuit protection.

💧 PRESSURE WARNING

This system operates under pressure. Ensure all pressure fittings are properly rated and installed. Install pressure relief valves. Never exceed maximum rated pressures.
